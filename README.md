### 🛠️ Proyectos Destacados

#### ☕ [Analizador-Lexico-Sintactico](https://github.com/fergusan98/Analizador-Lexico-Sintactico)
Desarrollé **desde cero y sin librerías externas** un analizador léxico y sintáctico para un lenguaje de programación propio en Java. 

* **Manejo de Autómatas:** Implementación manual de la lógica de tokenización (Lexer) para identificar palabras clave, operadores y literales.
* **Análisis Sintáctico:** Construcción del Parser (mediante descenso recursivo) para validar la gramática del lenguaje.
* **Ciencia de la Computación Pura:** Aplicación práctica de conceptos avanzados de teoría de lenguajes formales y estructuras de datos arbóreas.
